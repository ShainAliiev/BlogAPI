# Blog API

- This is an example of a Blog API.

- The application is contained within "BLOGAPI" folder.

### Install

- Doesn't need installation

### Run app

- Install and launch 'Postman Application'.

- Use command "nodemon server.js" in VS Code.

### NPM packages used

- "bcrypt",
  "dotenv",
  "express",
  "express-validator",
  "helmet",
  "jsonwebtoken",
  "mongoose",
  "morgan"
